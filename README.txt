Description:
- This project contains the test suite for FT4233.
  FT4233 is FTDI/Bridgetek's latest USB to Configurable Quad Serial Interfaces (UART, SPI, I2C, JTAG).

Directories:
- CMD
  This folder contains the commandline C/C++ application
- GUI
  This folder contains the Java Swing user interface for the commandline C/C++ application
- BIN
  This folder contains the official releases

Development enviroment:
- Windows 7 SP1

Software required/used:
- Microsoft Visual Studio Community version (https://www.visualstudio.com/free-developer-offers/)
- Eclipse Neon 3 Release 4.6.3 (https://www.eclipse.org/downloads/packages/release/Neon/3)
  
Devices required/used:
- FTDI MPSSE Cable C232HM-DDHSL-0 FT232H (http://www.ftdichip.com/Support/Documents/DataSheets/Cables/DS_C232HM_MPSSE_CABLE.PDF)
- FTDI Programmer Module UMFTPD2A FT4232HL (http://www.ftdichip.com/Support/Documents/DataSheets/Modules/DS_UMFTPD2A.pdf)
- Texas Instrument I2C EEPROM 24LC024H (http://ww1.microchip.com/downloads/en/DeviceDoc/22102A.pdf)
- Texas Instrument SPI EEPROM 93LC56B (http://www.microchip.com/wwwproducts/en/93LC56B)
- Winbond SPI Flash W25Q128JV (https://www.winbond.com/resource-files/w25q128jv%20spi%20revc%2011162016.pdf)
